## Introduction

App2Docker is a visual tool for converting local applications or runtime environments into Docker images, suitable for UI-driven build and packaging workflows.

## Features

- Configure image builds through a web UI instead of writing complex Dockerfiles by hand.
- Push generated images to a registry or run them locally after the build completes.
- Persist build data and configuration in mounted directories for easier maintenance.
- Access the host Docker engine directly for image build operations.
