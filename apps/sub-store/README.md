# Sub-Store

高级订阅管理工具，适用于 macOS、iOS、Android 及服务端。

### 主要功能
- **多订阅合并**：将多个机场订阅合并为一个。
- **协议转换**：支持 VLESS, Reality, Hysteria2 等最新协议。
- **规则独立**：通过 External Config 实现节点与分流规则的完全分离。
- **脚本增强**：支持使用 JavaScript 对节点进行重命名、筛选和排序。

### 安装说明
安装完成后，若设置了后端路径（如 `/clash`），请通过以下方式访问：
`http://服务器IP:端口?back-end=/clash`
