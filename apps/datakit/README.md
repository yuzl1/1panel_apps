# openGauss DataKit 可视化平台

openGauss DataKit 是一款集安装、运维、监控和开发于一体的可视化管理平台。通过该平台，你可以屏蔽复杂的命令行操作，实现对 openGauss 数据库的图形化管理。

## 🚀 快速启动指引

### 1. 访问地址
- **URL**: `https://服务器IP:9494`
- **注意**: 必须使用 **HTTPS** 协议访问。由于使用的是自签名证书，浏览器会提示安全风险，请点击“高级”并选择“继续访问”。

### 2. 初始登录凭据
- **默认用户**: `admin`
- **默认密码**: `admin123`
- **首次登录**: 系统会强制要求修改初始密码，请务必设置并记住新密码。

### 3. 关键配置参数 (AES_KEY)
- 安装时设置的 **AES_KEY** 用于平台内部数据的加解密（如数据库连接密码）。
- **警告**: 请妥善保管此 Key。如果未来需要重新安装或升级，必须使用相同的 Key，否则会导致加密数据无法读取。

## 🛠️ 运维与管理

### 后台数据库
本应用默认使用内置的 **Intarkdb** 数据库。如果你需要切换为外部 openGauss 数据库，请修改 `data/config/application-temp.yml` 并重启应用。

### 目录说明
- `data/data`: 存储内置数据库文件。
- `data/logs`: 存储系统运行日志。
- `data/config`: 存储 DataKit 的核心配置文件。

### 常见问题
- **端口连接失败**: 确保宿主机防火墙已放行 9494 端口。
- **权限报错**: 若容器无法启动，请检查宿主机挂载目录的权限，建议执行 `chmod -R 777 data`。
