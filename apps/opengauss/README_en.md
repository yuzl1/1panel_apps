## Introduction

openGauss is an open source relational database management system designed for self-hosted deployments that require performance, availability, security, and maintainability.

## Features

- Provide an openGauss database runtime with persistent storage.
- Configure administrator username, password, and service port during installation.
- Include a health check for easier service observation in 1Panel.
- Support connections from tools such as `gsql`, DBeaver, and Navicat.
