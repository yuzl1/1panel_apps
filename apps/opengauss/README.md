# openGauss

openGauss 是一款开源关系型数据库管理系统，采用木兰宽松许可证 v2 发行。openGauss 内核深度优化了多核架构、智能索引、列式存储等技术，具备高性能、高可用、高安全、易运维等特点。

## 快速入门

1. **配置参数**：
   - **数据库管理员密码**：必须满足复杂度要求（大写+小写+数字+特殊字符）。
   - **服务端口**：外部访问数据库的端口，默认为 `5432`。
2. **连接数据库**：
   - 使用 `gsql` 或 `Navicat`、`DBeaver` 等工具。
   - 用户名默认为 `gaussdb` 或 `omm`（取决于具体镜像版本，通常社区镜像为 `gaussdb`）。

## 注意事项

- **数据持久化**：应用数据默认挂载在宿主机的 `data` 目录下。
- **系统资源**：openGauss 对内存有一定要求，建议分配至少 2GB 以上内存。
- **初始化时间**：首次启动时进行数据库初始化较慢，请通过 1Panel 日志查看进度，看到 `database system is ready to accept connections` 后即可连接。
